### hexo init 卡住

https://www.jianshu.com/p/a409601734f1
![图片](gitee-hexo/1.png)

```
hexo init blog
```

换成, (下面 npm 换成 cnpm 更快)

```
git clone --depth 1 https://gitee.com/weilining/hexo-starter.git blog
cd blog
git submodule init
git submodule update
cnpm config set registry http://registry.npm.taobao.org
cnpm i
```
